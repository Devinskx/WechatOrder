# WechatOrder
微信点餐服系统开发（其中的微信支付功能没有写全，因为企业级别的微信公众号才有微信支付的接口，其实按照微信开发平台所给的接口开发就可以，很简单）

主要使用到的技术有Spring Boot+hibernate/Mybatis+MySQL+Redis

总体设计：
  主要分为买家和卖家两个大的方面进行设计，买家端，也就是微信端，能实现生成订单，查看商品列表以及单个商品详情，提交订单，微信支付（因为客观原因未完成），取消订单灯功能
  而卖家端则可以在后台查看数据，可以查看订单，按照客户要求取消订单，并且能够自己管理商品包括增删查改。为了实现卖家端后台管理的身份唯一性，使用了redis缓存和cookie来共同实现身份的验证。
